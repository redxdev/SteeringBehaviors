The Framework

Provided inside the unity project is a basic framework intended to make it very easy to play with steering behaviors. Inside the /SteeringBehaviors/Scripts/ folder, there is a SteeringManager script. This is what handles actually moving your agents around the world. To use it, you also need a Character Controller component. You can see how an agent might be set up in the inspector to the right. An alternative to setting up an agent yourself, you may also use the /SteeringBehaviors/Prefabs/basic steering setup prefab, which already has all the components you need.

Basic Concepts

Force

Force is the primary way to move agents around. Whenever a steering behavior needs to move an agent, it tells the steering manager to apply a force. The important thing to note here is that a steering behavior should never apply the force itself.

Weight

Weight is the ratio of one steering behavior's force to another. When force is applied, it is weighted relative to the total weight of forces that frame.

For example, given the forces \(\vec{a}\) with weight 1 and \(\vec{b}\) with weight 2, force \(\vec{b}\) will be twice the magnitude of \(\vec{a}\).

Writing Behaviors

Writing behaviors is incredibly easy. Here we'll use C# (though this should work fine in JS as well), so create a new C# script (henseforth known as MyBehavior.cs). Steering behaviors are, in fact, just normal Unity components which simply call AddForce(weight, force) on a SteeringManager object. Start with the following code: 

using UnityEngine;

[RequireComponent(typeof(SteeringManager))]
public class MyBehavior : MonoBehavior
{
    private SteeringManager steering = null;

    void Start()
    {
        steering = GetComponent<SteeringManager>();
    }
}

That's all the boilerplate code you need to set up a steering behavior. Next, we'll apply a force. Create the following method in your class. 

void Update()
{
    steering.AddForce(1, new Vector3(1, 0, 0));
}

You may also want to add a CharacterController field such that you can retrieve information about velocity, but that isn't required for this tutorial.

You're done creating your first (simple) behavior. The most important line of the entire class here is the steering.AddForce(1, new Vector3(1, 0, 0)); line, which tells the steering manager to apply a force. The first argument is the weight of the force. The second argument is the actual force you want to apply.

That's it for how to use the framework. Take a look at the other examples and tutorials, or look through the source code to see how everything works!

When applying a force via SteeringManager.ApplyForce, make sure that the force you are applying is normalized. Otherwise, the weight system may be thrown off depending on how far from normalized the force is.
