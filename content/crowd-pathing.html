Going Further

This demo shows off a combination of flocking and path following behaviors. The prefab agent is /SteeringBehaviors/Prefabs/crowd, and the example scene is /SteeringBehaviors/Scenes/crowd pathing.

This concludes the behavior and framework tutorials. This is a demo that uses the previous behaviors in new and interesting ways.

Crowd Path Following

To simulate a group of people following a path, we can use both the path following and flocking behaviors on a single agent. After playing around with the weights and other settings, you end up with a demo such as above.
